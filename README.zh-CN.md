# MCT

[English](README.md) | 简体中文

MCT 是革命性的Minecraft硬编码地图的翻译工具，它彻底改变了传统性的翻译流程。

不同于传统性质的手工翻译， MCT简化了文本的提取与回填，使得翻译效率显著提升，版本迁移成本显著降低。

## 特性
- 📦 自动化的文本提取与回填
- 🤖 内置的AI翻译支持
- 📝 专门为手工汉化者设计的MTLX DSL
- 🎯 支持数据包、区块文件以及其他的资源
- 🔌 可扩展性的自定义提取规则
- 😇 用户友好的GUI页面
